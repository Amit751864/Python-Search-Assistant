# 🔍 Python Search Assistant

A simple desktop application built with **Python** and **Tkinter** that allows users to quickly search on multiple websites directly from a single GUI.

---

## 📌 Features

- 🔍 Google Search
- ▶️ YouTube Search
- 📷 Instagram Profile Search
- 👥 Facebook Search
- Simple and beginner-friendly GUI
- Opens results in your default web browser

---

## 🛠️ Built With

- Python 3
- Tkinter
- Webbrowser Module

## 🚀 How to Run

### 1. Clone the repository

```bash

```

### 2. Move into the project folder

```bash
cd Search-Automation-GUI
```

### 3. Run the application

```bash
python main.py
```

## 📖 How It Works

1. Enter a search query or Instagram username.
2. Click the desired platform button.
3. The application opens the corresponding website in your default browser.
